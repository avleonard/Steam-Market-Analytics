# Steam Data Analysis Projects

Using historical sales data directly from Steam's API, these projects analyze platform evolution, performance metrics, and pricing dynamics through statistical analysis and machine learning.

> **Technologies:** *Python, Pandas, Scikit-learn, Matplotlib, Seaborn*

### Project 1: Analyzing How Steam Has Matured
- Analysis of Steam's change over time using transaction data from their API. Examines annual game releases, pricing trends, rating evolution, and platform support (Windows, macOS, Linux) from 2006 to 2025.

### Project 2: Predicting Steam Game Review Scores
- Using a Random Forest, predict review categories (High/Medium/Low) from attributes like reviews, current price, current discount, and release date.

### Project 3: Predicting Steam Game Prices
- Using the best performing regression model (Linear, Ridge, Random Forest, Gradient Boosting), predict prices with reviews, ratings, discounts, and platform support data.
